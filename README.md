# Study-Blog

![이미지 오류](image.png)

### 프로젝트 소개

---

- 플랫폼: Web / 웹
- 제작인원: 3 인 ( 팀 구성: Frontend 1명, Backend 2명 )
- 제작 기간: 30 일 (2024.10.01 ~ 2024.31.)
- 사용 기술: React, Express

'Study-Blog'는 공부에 대한 지식을 필요로 하는 사람들의 대한 웹입니다.
기존에 공부자료가 흩어져 있어 찾기 불편했다면 접근성이 용이한 웹으로 언제 어디서든 접속하여 필요한 공부자료를 찾아볼 수 있습니다.
